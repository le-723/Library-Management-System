# Library-Management-System
图书管理系统
基于Spring+Spring MV

项目简介
本图书管理系统基于spring,spring mvc,数据库为mysql。前端使用了Bootstrap。

系统功能
该系统实现读者和管理员登录，管理员在系统中可以实现图书管理、读者管理、借还管理以及修改密码，读者在系统中可进行查询图书、借阅图书、归还图书、查看个人信息和修改密码
